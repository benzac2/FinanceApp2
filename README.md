# FinanceApp2 - Aplikacja do zarzadzania finansami osobistymi

WinForms (.NET 10.0) + SQLite

## Wymagania

- .NET 10.0 SDK - https://dotnet.microsoft.com/download/dotnet/10.0

## Uruchomienie

```bash
git clone https://github.com/benzac2/FinanceApp2.git
cd FinanceApp2
dotnet restore
dotnet build
dotnet run
```
